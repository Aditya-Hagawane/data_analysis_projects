👥 Employee Attrition Analytics using Python
📌 Project Objective

To analyze employee attrition patterns using exploratory data analysis (EDA) in Python and identify factors associated with employee turnover.

📊 Project Type

Descriptive and diagnostic analytics project (EDA-based).

📂 Data Source

Employee Attrition Dataset (CSV)

🛠 Tools Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

📈 Project Features

Data loading and cleaning

Exploratory data analysis

Distribution analysis

Group-based analysis

Comparative analysis (Attrition vs Non-Attrition)

Data visualization

Pattern identification

🔍 Business Problems Addressed

Which departments show higher attrition?

Which job roles show higher attrition?

How does attrition vary with job satisfaction?

How does attrition relate to income levels?

How does attrition vary with work-life balance?

How do age and experience affect attrition patterns?

Which workforce segments show concentrated attrition?

📌 Key Insights (EDA-Based)

Attrition is not evenly distributed across departments.

Certain job roles show higher attrition concentration.

Lower job satisfaction is associated with higher attrition presence.

Income distribution differs between attrition and non-attrition employees.

Work-life balance shows variation between attrition groups.

Attrition patterns are clustered within specific employee segments.

🩺 Recommendations (Data-Supported)

Focus on high-attrition departments for retention discussions.

Review job roles with higher attrition concentration.

Address workplace factors linked to employee dissatisfaction.

Use segmentation analysis for workforce planning discussions.

Monitor high-risk workforce segments through periodic analysis.
