# 📊 Data Analytics Portfolio

This repository showcases a collection of end-to-end data analytics projects using **Python, SQL, Excel, and Power BI**.  
The projects focus on **data cleaning, exploratory data analysis (EDA), dashboard development, and generating actionable business insights**.

---

## 🚀 Projects Included

### 1. Employee Attrition Analysis (Python)
- Performed data cleaning and exploratory data analysis (EDA) on employee dataset  
- Identified key factors affecting employee attrition such as salary, job role, and work environment  
- Used Python (Pandas, NumPy, Matplotlib, Seaborn) for data manipulation and visualization  
- Generated insights to support HR decision-making and reduce employee turnover  

---

### 2. Healthcare Dashboard
- Analyzed healthcare dataset to identify trends in patient data and hospital performance  
- Performed data preprocessing and feature analysis  
- Built an interactive dashboard to visualize KPIs and healthcare insights  
- Enabled better decision-making through data visualization  

---

### 3. Store Sales Analysis (Python & Dashboard)
- Analyzed sales data to identify revenue trends, top-performing products, and seasonal patterns  
- Performed data cleaning and EDA using Python  
- Built dashboard to track KPIs such as revenue, profit, and sales performance  
- Provided insights for improving sales strategy and business growth  

---

### 4. Banking Data Analysis (SQL)
- Analyzed banking dataset using SQL queries (joins, aggregations, subqueries)  
- Performed data analysis 
- Identified customer trends, transaction patterns, and key financial insights  
- Generated reports to support business decision-making  

---

## 🛠️ Tools & Technologies
- **Python:** Pandas, NumPy, Matplotlib, Seaborn  
- **SQL:** MySQL / PostgreSQL (Joins, Aggregations, Subqueries)  
- **Visualization:** Power BI, Excel Dashboards  
- **Other:** Jupyter Notebook, Git
